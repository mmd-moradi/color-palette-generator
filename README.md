# color-palette-generator
Color palette generator app
--------------
![final-img](https://github.com/mmd-moradi/color-palette-generator/assets/110211422/46c6eba2-4d4d-4607-8081-26c1ba255a35)
---------
The idea behind this project is to generate random 5 different color palettes with the functionality to save them.
This project was built only by HTML, CSS and Vanilla JS.
